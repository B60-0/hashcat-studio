# First Run

1. Open Hashcat Studio.
2. Click Get Started.
3. Choose one setup option:
   - Download Hashcat
   - Choose a Hashcat folder
   - Choose a Hashcat binary
   - Use `hashcat` from PATH
4. Wait for setup to validate Hashcat.
5. Add your hashes and wordlists.
6. Create a task and preview the command.
7. Check Devices to confirm Hashcat can see your CPU or GPU devices.

Hashcat Studio creates folders for:

- hashes
- dictionaries
- rules
- masks
- output

You can change those folders in Settings.
