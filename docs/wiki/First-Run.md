# First Run

1. Open Hashcat Studio.
2. Go to Settings.
3. Set the Hashcat binary path.
4. Click Validate.
5. Confirm algorithms load.
6. Check Devices to confirm Hashcat can see your CPU or GPU devices.

Hashcat Studio creates folders for:

- hashes
- dictionaries
- rules
- masks
- output

You can change those folders in Settings.
